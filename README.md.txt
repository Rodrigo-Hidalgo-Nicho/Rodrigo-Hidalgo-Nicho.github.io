•	Color: El color predominante es el blanco. Para el landing page, se optó por una tonalidad más suave que no genere distracción en el usuario. Para representar las variaciones dentro de diseño se incluyeron las dos variantes de colores elegidos para la marca, el negro y el blanco. Este contraste de color destaca las distintas secciones del landing page. Se opta por un diseño suave y sencillo.
•	Space: Se ha considerado el empleo adecuado del espacio para una mejor identificación de las secciones, para ello se usó boxes que permiten separar el contenido, como en las secciones “¿Cómo funciona?” o “Beneficios”. 
•	Shapes: Dentro de las figuras utilizadas para representar las imágenes o contenido, se aprecia el uso de rectángulos, círculos e imágenes para una mejor apreciación de los servicios y contenido. De igual manera, los íconos también son empleados durante el diseño y tamaños para los elementos dentro del diseño.
•	Alignment: Se destaca la alineación de los boxes dentro del diseño para ofrecer una visualización ordenada durante el recorrido del landing page.
•	Contraste: Permite que se puedan apreciar todos los componentes del landing para beneficio del usuario. Estas tonalidades de gris, nrego y blanco se puede apreciar en los textos, figuras, servicios, entre otros. 
•	Para los patrones web se usó navegación (jump to section, call to action, vertical navigation), patrón F debido a que el diseño del landing guía al usuario a recorrer las secciones de manera ordenada.
•	Dentro de las heurísticas de arquitectura de información destacan “findable” para que el usuario encuentre lo que busca gracias al header. Accesible, con un diseño responsive que se adecua a cualquier dispositivo. Learnable, su contenido es fácil de entender. Usable, debido a que se ubicaron varios call to action en las secciones del landing page
